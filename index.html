hello hexo

本文记录安装hexo，并在github.io上建blog的过程。

系统：utuntu 17.10

0,参考
nvm：https://github.com/creationix/nvm
hexo: https://hexo.io/zh-cn/docs/index.html
hexo建站: https://www.jianshu.com/p/4eaddcbe4d12

1,安装nvm

curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.11/install.sh | bash

问题1: 装完会报错，找不到nvm。
解决：编辑.bashrc
增加 source ~/.nvm/nvm.sh,重启terminal。

2,安装node
nvm install stable

3,安装hexo
npm install -g hexo-cli

4,基本环境准备结束，下面开始使用hexo 创建站。以下命令在当前目录创建有一个 reneix.github.io，是我们站点的根目录

hexo init reneix.github.io
cd reneix.github.io
npm install

编辑_config.yml,修改语言和部署选项

language: zh-Hans
deploy:
type: git
repo: https://github.com/reneix/reneix.github.io.git

5,创建一篇文章，名叫helloword，物理路径在 reneix.github.io/source/_posts/helloworld.md

hexo new helloworld

6,编辑之后本地启动服务预览

hexo server

会在localhost:4000 启动web服务，浏览器打开即可预览效果。
